# JavaScript Mini Projects

A collection of 5 beginner JavaScript projects built with vanilla HTML, CSS, and JavaScript.

---

## Projects

### 1. 🎨 Colour Flipper
Changes the background colour of the page using preset colour buttons or a random colour generator.

**Features:**
- Green, Red, and Blue preset buttons
- Random colour generator using RGB values

**Files:** `min.html`, `index.js`

---

### 2. 🔤 Palindrome Checker
Checks whether a word or phrase entered by the user is a palindrome.

**Features:**
- Alerts the user if the input is a palindrome or not
- Clears the input field after each check

**Files:** `main.html`, `index.js`

---

### 3. 💬 Random Quote Generator
Displays a random motivational quote from a preset list, ensuring no quote repeats until all have been shown.

**Features:**
- 10 motivational quotes
- Tracks used quotes to avoid repetition

**Files:** `main.html`, `index.js`

---

### 4. ⏱️ Stopwatch
A functional stopwatch with start, stop, and reset controls.

**Features:**
- Displays time in `MM:SS` format
- Start, Stop, and Reset buttons
- Prevents duplicate intervals on repeated starts

**Files:** `main.html`, `index.js`

---

### 5. ✅ TODO List
A simple to-do list app that persists items across page reloads using localStorage.

**Features:**
- Add and delete items
- Data saved to `localStorage`
- Items load automatically on page open

**Files:** `main.html`, `index.js`

---

## How to Run

Each project is self-contained. To run any project:

1. Clone or download this repository
2. Navigate to the project folder
3. Open `main.html` (or `min.html` for Colour Flipper) in your browser

No installations or dependencies required.
